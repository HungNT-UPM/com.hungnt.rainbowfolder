# HungNT Rainbow Folders (`com.hungnt.rainbowfolder`)

Plugin editor **Rainbow Folders** (Borodar) — tô màu / icon folder trong Project window.

> **License Asset Store** — không có trên OpenUPM.

## Sử dụng

1. Import package (UPM hoặc copy plugin Borodar vào `Assets/Plugins/Borodar` nếu dùng layout legacy).
2. Right-click folder trong Project → **Rainbow Folders** → chọn màu / preset.

Chỉ chạy trong **Unity Editor** — không ảnh hưởng build runtime.

QuickStart / PDF trong plugin gốc Borodar (`NOTICES.txt`).
